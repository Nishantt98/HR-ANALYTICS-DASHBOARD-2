# HR-ANALYTICS-DASHBOARD-2

This HR Analytics Dashboard provides a comprehensive visual analysis of employee data, focusing on attrition patterns, workforce demographics, and job satisfaction metrics. It is designed to help HR professionals and stakeholders identify trends and drivers behind employee turnover.

4. Job Satisfaction & Roles
A detailed matrix (Heatmap/Table) correlates Job Roles with Satisfaction Scores (1 to 4). 
Roles included: Laboratory Technician (259 total), Sales Executive (326 total), Research Scientist (292 total), and more. 
The table helps identify roles with high "Level 1" satisfaction scores, which may correlate with higher attrition.
